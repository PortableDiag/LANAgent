VR Mode
View these visualizations in VR with any WebXR headset (Valve Index, Quest, etc.).

Setup: WebXR requires HTTPS. For local network access, add your server URL to:
chrome://flags/#unsafely-treat-insecure-origin-as-secure

Then restart Chrome with SteamVR running and the "Enter VR" button will appear.
